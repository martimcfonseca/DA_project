Introduction

This project is a submission-reviewer assignment system. It reads submission and reviewer data from CSV files, checks if it is possible for every submission to be reviewed by the minimum required number of reviewers, and also allows performing a risk analysis in case a reviewer fails.

Usage

  • Run the program in the terminal.
  • Load input files through the menu.
  • List submissions, reviewers, or parameters (if you want to see).
  • Run the assignment computation and optionally perform risk analysis.

Code Structure

The project is organized as follows:

  • Input.h/cpp - handles reading CSV files and storing submissions, reviewers, and parameters.
  • Graph.h/cpp - defines the graph structures.
  • MaxFlow.h/cpp - implements the Edmonds-Karp algorithm and outputs results.
  • RiskAnalysis.h/cpp - performs critical reviewer analysis.
  • Parser.h/cpp - parses CSV files and builds the graph.
  • Menu.h/cpp - displays the terminal menu and manages user actions.
  • main.cpp - entry point of the program.
